# AI Prompt Validator for GenLayer

## Description
This tool analyzes smart contract prompts and detects ambiguity, missing logic, and unsafe instructions.

## Features
- Detects vague language
- Flags missing transaction details
- Suggests improvements

## How to Run
1. Install Python
2. Run: python app.py
3. Enter your prompt

## Example
Input:
"Send ETH if conditions are met"

Output:
- Ambiguity warning
- Suggestion for improvement

## Use Case
Helps GenLayer developers write safer and clearer AI-driven smart contract instructions.
